## Бот для отправки голосовых оскорблений в Telegram!

# Требования:
- Python 3
- Inline mode у твоего бота
- Пак козырных звуков

# Установка:
- Клонируй репо и создай файл .env с переменными BOT_KEY и CHAT_ID
- Установи зависимости
- Свои шедевральные оски кидай в папку zvuchok, после чего выполняй upload.py, ну или можешь в диалог с ботом отправить файл ogg opus с нужным названием. Ток права на файлы на сервере не забудь проставить.
